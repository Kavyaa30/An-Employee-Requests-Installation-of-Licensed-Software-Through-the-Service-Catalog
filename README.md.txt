# Software Request Automation (ServiceNow)

## Overview

This project automates software installation requests using ServiceNow Flow Designer.

## Flow Steps

1. Trigger: Requested Item Created
2. Ask for Approval
3. If Approved → Create Catalog Task

## Features

* Automated approvals
* Task creation
* Reduced manual work

## Issues Faced

* Flow stuck in waiting
* Fixed approval and data mapping

## Result

Automation of software request lifecycle successfully implemented.
